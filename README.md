# 🛠️ Mains Repository

## 📌 Description
Repository containing `main` files for testing C projects from the 42 cursus.

## 📂 Contents
- `main_libft` → Tests `libft`
- `main_gnl` → Tests `get_next_line`
- `main_ft_printf` → Tests `ft_printf`

## 🚀 Usage
```sh
$ gcc main_libft.c -L../libft -lft -o test_libft
$ ./test_libft
```
